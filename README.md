## 🚀 Sobre o Projeto

Este é meu **portfólio pessoal** — um espaço onde apresento quem eu sou, minhas habilidades e os projetos que já construí.

**O que você vai encontrar por aqui:**

- 🧑‍💻 Apresentação profissional com foto e bio
- 🎨 Tema claro/escuro com alternância dinâmica
- ⚡ Quiz interativo "Descubra seu perfil dev"
- 🗂️ Seção de projetos com cards dinâmicos
- 📅 Timeline de formação acadêmica
- 📱 Layout responsivo para todos os dispositivos

## ✨ Funcionalidades em Destaque

### 🌗 Tema Dinâmico
Alternância entre modo **claro** e **escuro** com um clique, sem reload de página.

### 🎯 Quiz de Perfil Dev
Descubra se você tem mais perfil de dev **visual** (front-end focado em design) ou **lógico** (back-end e dados). Interativo e divertido!

### 📊 Timeline Acadêmica
Datas de formatura e ingresso renderizadas dinamicamente via JavaScript — sempre atualizadas.

### 🗂️ Projetos Dinâmicos
Cards de projetos carregados via script, fáceis de atualizar sem mexer no HTML.
